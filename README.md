# TemplateShow

This folder is a template for creating new Fins and Pins Click-to-Claim shows.

## How to use this template

1. Duplicate this entire folder in the Files app or Working Copy.
2. Rename the duplicated folder to the new show name.
3. Replace the images in /boards/ with the new board JPEGs.
4. Replace the JSON files in /boards/ with the new overlays.
5. Update manifest.json to list the board filenames (without extensions).
6. Edit index.html and update the show title in the <h2> tag.
7. Commit and push to GitHub.

## Folder structure

TemplateShow/
    index.html
    boards/
        manifest.json
